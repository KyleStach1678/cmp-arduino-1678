Citrus Arduino Library

This library contains all of the necessary elements for the Citrus Circuits offseason Arduino competition. To download it, install git and use the following commands in git bash:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Alternatively, click download .zip above

Installation

First, install the Arduino IDE by clicking the download link for your computer and going through the wizard. If you downloaded the files in a zip file, unzip it into a directory. Open the main.ino file and start programming.

Drivers

To upload code onto the Arduino, you will need to install the drivers for communication. Refer to this guide for instructions.
